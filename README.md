# Akash Jain — Portfolio

This repository contains a responsive portfolio website and a print-friendly resume page.

Preview locally:

1. Open `index.html` in your browser (no server required).
2. Or run a simple local server (Python):

```powershell
python -m http.server 8000
# then visit http://localhost:8000
```

Deploy to GitHub Pages:

1. Create a repo on GitHub and push this folder as the repository root.
2. In repo settings -> Pages, select the `main` branch and `/ (root)` as the folder.
3. Your site will be available at `https://<username>.github.io/<repo>`.

Notes:
- `resume.html` is optimized for printing to PDF (use browser Print -> Save as PDF).
- If you want a compiled PDF from `aj_resume.tex`, compile locally with `pdflatex aj_resume.tex` and add the PDF to the repo.
